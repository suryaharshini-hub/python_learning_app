# PyVerse – Python Learning Platform

PyVerse is a beginner-friendly interactive coding platform designed to help users understand fundamental Python concepts through practice.

The platform focuses on core Python data structures and patterns while providing an in-browser coding environment where users can test and experiment with code instantly.

## Live Demo

https://suryaharshini-hub.github.io/python_learning_app/

## Features

### Interactive Code Editor

Users can write and run code directly inside the browser.

Example default code provided in the editor:

```
let arr=[1,2,3,4];
arr.forEach(x=>console.log(x));
```

Example learning scenario:

* User edits the code
* Clicks **Run Code**
* Output appears instantly in the console

---

### Python Concepts Covered

The platform currently supports learning the following topics:

* Python Lists
* Python Tuples
* Python Sets
* Pattern Problems

Example learning flow:

1. Select a topic from the sidebar
2. Read the short explanation
3. Modify the default code
4. Run and observe the output

---

### Default Starter Code

Each topic automatically loads a default code example to help beginners understand syntax quickly.

Example:

* List iteration example
* Pattern printing structure

This reduces the confusion beginners face when starting from a blank editor.

---

### Error Feedback with Beep Sound

If the code contains an error during execution, the platform produces an audio beep to immediately notify the user.

Example situations:

* Syntax mistake
* Invalid statement

This improves the learning feedback loop.

---

### Learning Streak System

The platform tracks user streak activity to encourage consistent learning.

Example:

* Day 1 → user practices list exercises
* Day 2 → user continues with tuples

Maintaining streaks helps build regular coding habits.
